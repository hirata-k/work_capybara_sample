# work capybara sample

## Required

- jdk 16
- chromedriver
- ruby 3.0.0

## Usage

### download selenium server

https://www.selenium.dev/downloads/

### run selenium server

```
java -jar selenium-server-4.0.0-beta-4.jar standalone
```

### run sample

```
bundle install
```

```
bundle exec rspec example.rb
```
